# PRODUCT.md — 产品定义

## 愿景

做一个**自己每天都会用**的 AI 工作台。不做大而全，只做高频刚需。

## 目标用户

- 我自己
- 任何想用 AI 提升日常效率的人

## 核心原则

### 极简
- 功能少而精，每个功能都经高频使用验证
- 能用一个按钮解决的，不用两个

### 可组合
- 脚本之间可以串联，形成工作流
- 输入输出标准化，像搭积木一样

### 本地优先
- 数据存在本地，不上传第三方
- 离线也能用核心功能

## 功能规划

### V1 — 基础工作台
- [ ] AI 对话终端（多模型切换）
- [ ] 提示词模板库
- [ ] 一键代码审查
- [ ] 文档速读与总结

### V2 — 自动化
- [ ] 定时任务调度
- [ ] 工作流编排（A → B → C）
- [ ] 结果通知推送

### V3 — 知识库
- [ ] 本地文档索引
- [ ] 对话历史检索
- [ ] 个人知识图谱

## 不做

- ❌ SaaS 服务
- ❌ 多人协作
- ❌ 复杂 UI
- ❌ 任何与"卖钱"有关的功能
