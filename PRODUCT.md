# PRODUCT.md — 产品定义

## 产品愿景

打造一个**全自动、零人工干预**的投研数据流水线，覆盖 A 股 + 美股 + 港股三市场，从原始数据采集到多模型估值到统一报告，全程自动化。

## 目标用户

- 个人投资者（深度价值投资风格）
- 需要跨市场数据对比的投研人员

## 核心功能

### 1. 数据采集层
- 美股：Finviz 批量采集（51 只核心标的 + ETF）
- A 股：腾讯财经行情 + Sina 现金流数据（100+ 标的）
- 港股：基础行情覆盖
- 宏观：猪周期 / 汇率 / 指数

### 2. 估值引擎
- DCF 两阶段模型
- 行业 PE 基准（光伏 14.5x / 养殖 18.0x / 锂电 22.0x / 饲料 16.0x / 疫苗 20.0x）
- PRE_REVENUE_TECH 特殊处理（PS 替代）
- FCF 负值自动修复（PB+PS 加权）

### 3. 修复引擎
- P0-1：美股 Finviz 全量财务
- P0-2：寒武纪 PRE_REVENUE_TECH
- P1-1：股本修正
- P1-2：FCF 负值 EV/EBITDA 替代
- P1-3：周期股分行业 PE

### 4. 报告输出
- `multi_valuation_all.md`：全市场全标的估值一览
- `stock_overview.md`：市场概览

## 非目标（v1.0 不做）

- 实时交易信号
- 技术面分析
- 回测系统
- Web UI
