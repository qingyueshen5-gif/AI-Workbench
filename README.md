# AI Workbench

一个极简的 AI 工作台。

## 是什么

把 AI 工具、脚本、自动化流程整合到一个地方，让你用最少的操作完成日常 AI 任务。

## 能做什么

- 代码生成与审查
- 文档写作与翻译
- 信息检索与总结
- 自动化工作流编排

## 快速开始

```bash
git clone https://github.com/qingyueshen5-gif/AI-Workbench.git
cd AI-Workbench
```

## 目录结构

```
AI-Workbench/
├── scripts/      # 自动化脚本
├── prompts/      # 提示词模板
├── tasks/        # 任务记录
└── docs/         # 文档
```
