# AI Workbench v0.4.6 Alpha

AI Workbench v0.4.6 Alpha 是首批真实用户测试版本，仅面向 Windows 桌面端。

## 这版能做什么

- Windows 桌面版安装包。
- 安装后无需用户填写 DeepSeek API Key。
- 模型请求通过 AI Workbench Managed Proxy 调用 `deepseek-chat`。
- 支持安装、启动、真实对话、中文错误说明和卸载。
- 临时反馈方式：GitHub Issues。

## 使用条件

- 当前仅支持 Windows。
- 需要互联网连接。

## Alpha 边界

- 当前是 Alpha 测试版，不冒充成熟稳定版。
- 不包含手机端。
- 不包含模型分层。
- 不包含完整多 Agent 调度。
- 不包含跨网站自动执行。
- 不建议输入高度敏感或机密信息。
