# v0.1.1 显示面板扩充记录

## 做了什么

- 新增模型连接状态、用户偏好、存储状态三个首页显示区块。
- 新增系统级错误日志，并在历史页展示和搜索。
- 新增 DeepSeek API 单次连接测试入口。
- 扩展本地 JSON 数据结构，保存偏好、模型连接状态和系统错误。

## 为什么这样做

v0.1.x 仍处于功能显性版阶段，需要先把内部状态展示清楚，便于观察哪些逻辑有用，哪些应该在 Phase 3 合并、隐藏或删除。

## 下次改进方向

- 等待用户提供可用 API Key 后实测真实 DeepSeek API 连接。
- 不进入自动执行、多模型调度或偏好自动学习。
